# Landing Page Kit - Pixel Perfect

This project is a **Pixel Perfect** recreation of the "Landing Page Kit" interface, based on a design from the Figma community. 

The development focuses on extreme visual fidelity to the original design, adopting a **Mobile First** methodology and utilizing custom variables (typography, colors, and spacing) to ensure the coded result is a perfect mathematical reflection of what was designed.

## Original Design (Figma)
The layout and prototype that serve as the foundation for this project can be accessed here:
[Landing Page Kit - Free 13 Landing Pages Collection](https://www.figma.com/design/gCh9o5aU6agAjgLLxLH9sk/Landing-Page-Kit---Free-13-Landing-Pages-Collection-for-UI-UX-Design--Community---Copy-?node-id=366-35356&m=draw)

## Technologies Used
- **HTML5** - Semantic structure;
- **Tailwind CSS v4** - Used for core styling, with custom injected tokens and original CSS variables exported from Figma;
- **JavaScript (Vanilla)** - For page logic and interactions;
- **Vite** - Dev server and build tool for fast and optimized loading.

## How to Run the Project

1. Clone or download this repository.
2. Make sure you have Node.js installed on your machine.
3. Open the terminal in the project's root folder and install the dependencies:
   ```bash
   npm install
   ```
4. Then, start the development server:
   ```bash
   npm run dev
   ```
5. Access the local port indicated in the terminal (usually `http://localhost:5173`) to view the application!
